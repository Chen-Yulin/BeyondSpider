# BeyondSpider Space Combat

BeyondSpider is a Besiege mod context for capital-ship space combat, centered on energy allocation, sensor tracks, layered defense, and automated fire control.

Agent implementation guide: see `docs/agent-besiege-mod-guide.md` before adding or changing gameplay systems.

## Language

**舰船核心**:
一艘太空战舰的本地坐标和系统归属核心。它代表整船身份、能量预算，以及本舰被敌方雷达锁定时使用的空间锚点（位置/大小，不含朝向）。它不持有火控指挥UI或目标锁定权威，那是船长的职责；它的朝向也不代表整船朝向，那同样是船长的职责。
_Avoid_: 船长零件, 反应堆零件, 中控

**船长**:
全舰的火控指挥入口和目标锁定权威，同时**船长自身的朝向（right/forward）就是整艘太空战舰的朝向**——3D雷达图的网格坐标轴、导弹制导等一切需要"本舰朝向"的地方都以船长的 transform 为准，而不是舰船核心。呼出3D雷达图、在图上点击锁定/取消锁定目标都经由船长方块（`SpaceCaptainBlock`）进行；锁定结果是全舰共享的**火力通道**状态（四条通道各持一个锁定目标），供各武器按自己的通道订阅读取。船长不是舰船的空间锚点（位置仍由舰船核心代表），也不是雷达用来代表本舰的可被锁定位置（那是舰船核心的职责）。没有船长零件的舰船没有定义好的整船朝向，也无法呼出雷达图或锁定目标。
_Avoid_: 舰船核心, 中控, AA Captain

**火力通道**:
全舰共享的四条独立目标锁定槽位（0~3，颜色红/黄/青/紫），是武器与锁定目标之间的分组纽带。通道0是**防空通道**——船长在主机侧自动锁定接近速度/距离评分最高的敌对威胁，玩家可手动覆盖；通道1~3只由玩家在雷达图上点选。每条通道还各带一份**雷达筛选**，决定它关心哪几类目标。每个自动武器和炮手用 `MFireChannel` mapper 订阅任意通道组合；一个武器有多个候选目标时，通道0绝对优先，射界外的目标不参与，其余按 `1/(夹角+10°)×i + 1/距离` 评分。见 ADR-0010、ADR-0013。
_Avoid_: 锁定槽, 目标组, 武器编组

**能量预算**:
舰船核心在建造模式下分配给纳米装甲、护盾和武器的总能力。它是舰船规模、武装强度和质量惩罚的主要平衡量。
_Avoid_: 电量, 蓝条

**瞬时功率**:
某个系统在短时间内完成动作所需的功率压力。它决定超级电容是否需要介入缓冲。
_Avoid_: 耗电量

**超级电容**:
舰船系统的短时能量缓冲。它可以专供武器、护盾、装甲，或作为通用缓冲。
_Avoid_: 电池, 储能罐

**电磁场护盾**:
用于削弱或拦截高速实体弹体的防御层。它不防御激光。是否拦截以弹体**相对护盾发生器**的速度是否超过**超高速阈值**为准（**阈值判定用相对速度**）；相对速度越高，拦截消耗越大。它把弹体的**绝对**速度削向一个绝对下限（**减速目标用绝对速度**），而不是把弹体完全停住——炮弹和导弹共用同一套门槛与下限。
_Avoid_: 护盾

**超高速阈值**:
电磁场护盾开始削弱一发弹体所需的相对速度下限（600 m/s），以弹体相对护盾发生器的速度衡量。炮弹和导弹共用这一个门槛；低于它的弹体护盾不再处理。
_Avoid_: 600阈值, 拦截线

**纳米装甲**:
由舰体木头零件承载的自维护舰体防护，是一种**可击穿目标**。它分两层血条：可自修复的纳米层，与一旦受损就永不再生的结构层——结构层一旦被啃到，这块装甲就永久失去自愈能力。完好时强烈反射激光，破损后反射能力下降。
_Avoid_: 木头血量, 舰体 HP

**雷达面**:
面向指定锥形空域生成传感器轨迹的舰载侦察零件。它发现敌舰、大型炮弹和导弹。
_Avoid_: 雷达, 相控阵

**传感器轨迹**:
雷达面或其他传感器生成的目标记录。它描述目标位置、速度、类型、阵营、可信度和最后更新时间。轨迹是**某一次扫描的快照**，不是目标的实时状态：消费方（雷达图的光点插值、火控的提前量）要按最后更新时间推算目标当下在哪，而不能把轨迹位置当成此刻位置。
_Avoid_: 目标, 锁定物

**雷达筛选**:
一条火力通道关心哪几类可跟踪物体：舰船 / 重型导弹 / 中导弹 / 小导弹。它同时决定雷达图画哪些光点、以及这条通道能锁定什么——两者是同一份配置，不是两道独立的门。每条通道各有一份，玩家在雷达图顶部的筛选栏上按通道分别设置；通道0的自动防空也照此收窄候选。它不覆盖裸炮弹（没有对应的筛选行，因此不受它约束）。见 ADR-0013。
_Avoid_: 雷达过滤器, 显示开关, 目标类型菜单

**防空通道（Channel 0）**:
四条火力通道里编号 0 的那条，专职防空/点防御：舰长在主机侧持续按威胁评分自动锁定最高威胁的传感器轨迹（含不可锁定的裸弹体），玩家也可手动接管。所有防空/点防御武器（近防炮、点防御导弹、炮手）直接读取这条通道，不再有单独的拦截决策零件（ADR-0012 废弃了原来的"防御指挥官"方块）。它的候选同时受 IFF 和本通道的**雷达筛选**两道独立的门约束（前者管"是不是敌人"，后者管"这条通道关不关心这种东西"）。
_Avoid_: 防御指挥官, AA Captain, 防空雷达

**炮手**:
操控玩家自建炮塔铰链和炮管的自动化火控零件。它可以像单零件炮一样接收攻击命令。
_Avoid_: 自动炮塔

**炮塔铰链组**:
由一个炮手通过按键绑定接管的两类原版转向轮铰链集合：方位铰链组和俯仰铰链组。它描述绑定关系，不描述世界坐标或舰船坐标里的固定轴。
_Avoid_: 炮塔轴, 转台组

**单零件炮**:
单个零件即可完成瞄准、射击和伤害结算的武器。近防炮和单零件中口径炮都属于此类。
_Avoid_: AA

**单零件防空炮**:
用于防空/近防的单零件炮，当前对应 `SpaceFlakTurretBlock`。它通过自身视觉底座和炮管子物体运动来瞄准，不绑定玩家自建铰链。
_Avoid_: AA单零件炮, AA

**炮管武器**:
只提供发射管和弹道能力、需要玩家自建炮塔和炮手控制的武器。激光炮和电磁炮属于此类。
_Avoid_: 玩家炮

**火控命中点**:
火控系统预测出的世界空间瞄准点，炮塔把炮口指向它。因为弹体有**速度继承**，命中点以目标**相对本舰速度**的速度外推（目标速度 − 本舰速度），这样炮口在本舰参考系里正对拦截方向；提前量解算里的目标速度和外推都用这个相对速度。它不是目标当前位置，也不是传感器轨迹本身。
_Avoid_: 目标点, 提前量位置

**本舰速度**:
整艘太空战舰用于一切"相对速度"计算的单一线速度参考，取自**舰船核心**的刚体速度。火控把它当作所有炮台的线速度（不逐个查询炮塔速度），点防御的接近速度也以它为准。核心因此同时充当整船的位置锚点和线速度锚点，与代表整船朝向的**船长**分工；没有核心的舰船不参与这些相对速度计算。
_Avoid_: 船长速度, 炮塔速度, 每个发射器的速度

**速度继承**:
弹体发射时的初始世界速度 = 炮口朝向 × **武器初速** + 发射零件自身的刚体速度。电磁炮弹、防空炮大弹、拦截导弹发射时都叠加发射平台的速度，因此在高速移动的舰上开火，弹体会带着本舰速度飞出。瞬时命中的激光不涉及速度继承。
_Avoid_: 绝对初速, 世界初速

**武器初速**:
弹体相对于自己炮管的出膛速度（叠加到发射零件速度之上的那一份），不是弹体的世界速度。有了**速度继承**后两者不再相等，这正是火控必须在本舰参考系里算提前量的原因。
_Avoid_: 世界速度, 炮口世界速度

**相对速度**:
弹体（炮弹或导弹）世界速度减去某个参考零件速度后的速度。**电磁场护盾**的拦截阈值以弹体相对护盾发生器的速度衡量；点防御伤害惩罚、炮弹撞击动能伤害则以弹体相对被击中目标或**本舰速度**的接近速度衡量。参考零件是谁取决于是哪一种交互。
_Avoid_: 净速度

**炮弹（动能弹）**:
电磁炮和防空炮发射、在物理世界中飞行的实体弹丸。它以动能为唯一的毁伤本钱：命中装甲时被削减的动能，就等于对该装甲造成的毁伤（跳弹除外）；这份动能按弹体**相对被击中目标**的速度计，而不是它的世界绝对速度。因此一发炮弹可以连续穿过多块被清空血条的装甲，直到动能耗尽、跳弹、或被地形／尚未击穿的装甲挡下。它不再有与速度无关的"底伤"。
_Avoid_: 子弹, 平伤弹, 底伤

**可击穿目标**:
炮弹动能模型唯一会与之结算的实体：**纳米装甲**和**导弹**。它们都有"结构值"这条挡弹血条，炮弹按"剩余动能 ≥ 结构值"决定是否击穿并续飞。地形不属于它——地形是不可击穿的硬墙，直接销毁炮弹；船体上的非装甲零件也不属于它——炮弹从中穿过、毫无交互。是否可击穿与敌我无关。
_Avoid_: 硬目标, 碰撞体

**击穿**:
炮弹的剩余动能清空了一个**可击穿目标**的结构血条、并带着余下动能继续飞。装甲被击穿后除了断裂崩飞，还会对之后所有炮弹永久透明（不再挡弹）；导弹被击穿即销毁。动能不足以清空血条则不算击穿——炮弹要么嵌入停下、要么**跳弹**。
_Avoid_: 穿透, 洞穿, 破甲

**跳弹**:
炮弹未击穿装甲、且**命中角**大于 45° 时，按倾斜程度概率性地从装甲表面弹开的结果。它是"损失的动能不全等于毁伤"的唯一例外：一部分动能注入装甲成为毁伤、一部分被弹开耗散、炮弹自身仅保留少量动能改向续飞。只有装甲会跳弹，导弹不会。
_Avoid_: 弹飞, 反弹

**命中角**:
炮弹飞行方向与被击中装甲表面法线之间的夹角，0° 为正撞、90° 为擦过。它只决定**跳弹**的概率（大于 45° 才可能跳、越斜越易跳），既不改变毁伤大小，也不影响击穿判定。
_Avoid_: 入射角, 倾斜角
